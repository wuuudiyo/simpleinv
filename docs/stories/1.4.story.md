#### Button.tsx
```typescript
// src/renderer/components/ui/Button.tsx

interface ButtonProps {
  children: React.ReactNode;
  variant?: 'primary' | 'secondary' | 'danger' | 'ghost';
  size?: 'sm' | 'md' | 'lg';
  disabled?: boolean;
  loading?: boolean;
  onClick?: () => void;
  type?: 'button' | 'submit';
}

const variants = {
  primary: 'bg-blue-600 hover:bg-blue-700 text-white',
  secondary: 'bg-gray-100 hover:bg-gray-200 text-gray-700',
  danger: 'bg-red-600 hover:bg-red-700 text-white',
  ghost: 'hover:bg-gray-100 text-gray-700',
};

const sizes = {
  sm: 'py-1.5 px-3 text-sm',
  md: 'py-2 px-4 text-sm',
  lg: 'py-2.5 px-5 text-base',
};

export function Button({
  children,
  variant = 'primary',
  size = 'md',
  disabled = false,
  loading = false,
  onClick,
  type = 'button',
}: ButtonProps) {
  return (
    <button
      type={type}
      onClick={onClick}
      disabled={disabled || loading}
      className={`
        font-medium rounded-lg transition-colors
        disabled:opacity-50 disabled:cursor-not-allowed
        ${variants[variant]}
        ${sizes[size]}
      `}
    >
      {loading ? 'Laden...' : children}
    </button>
  );
}
```

### Design-Richtlinien [Source: PRD Section 3]

**Notion-inspiriertes Design:**
- Viel Whitespace (großzügige Abstände)
- Subtile Borders (`border-gray-200`)
- Keine schweren Schatten (nur `shadow-sm`)
- Sanfte Hover-Effekte (`hover:bg-gray-100`)
- System-Fonts für native Performance

**Farben:**
- Background: `bg-gray-50` (Light Mode)
- Cards: `bg-white`
- Text Primary: `text-gray-900`
- Text Secondary: `text-gray-500`
- Primary Action: `bg-blue-600`

**Status-Farben:**
- In Stock: Blau (`#3B82F6`)
- Listed: Orange (`#F59E0B`)
- Sold: Grün (`#10B981`)
- Returned: Rot (`#EF4444`)

### Coding Standards [Source: architecture/11-coding-standards.md]

| Element | Konvention | Beispiel |
|---------|------------|----------|
| React Components | PascalCase | `DashboardPage.tsx` |
| Component Files | PascalCase | `MetricCard.tsx` |
| Props Interfaces | PascalCase + Props | `MetricCardProps` |
| CSS Classes | Tailwind Utilities | `bg-white rounded-lg` |

**Regeln:**
- Eine Komponente pro Datei
- Index-Dateien für Re-Exports
- Keine inline styles, nur Tailwind-Klassen
- Props mit TypeScript Interface typisieren
